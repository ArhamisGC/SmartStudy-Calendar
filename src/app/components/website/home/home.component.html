Quiénes Somos

Somos un equipo de estudiantes de la Universidad de Gran Canaria. Nos hemos unido para desarrollar SmartStudy, una aplicación para gestionar los estudios. Hemos detectado problemas comunes entre los estudiantes a la hora de organizar su tiempo de estudio y queremos ofrecer una solución efectiva para mejorar su rendimiento académico.

Aplicación Web

SmartStudy es una aplicación web basada en la nube, lo que significa que puedes acceder a tus datos desde cualquier dispositivo con conexión a Internet. Con una interfaz intuitiva y funciones poderosas, te ayudamos a optimizar tu proceso de estudio para lograr el éxito académico.
